### 1. 文档解析器 (DocumentParser)

**文件位置**: `src/document_parser.py`

#### 功能清单
- ✅ **段落提取** (`extract_paragraphs`)
  - 提取所有段落文本
  - 获取段落样式信息

  - 文件大小（KB）
  - 文件路径

- ✅ **页面设置提取** (`get_page_setup`)
  - 上下左右边距（mm）
  - 页面宽度和高度（mm）
  - 页眉页脚距离（mm）

### 2. 字体工具 (FontUtils)

**文件位置**: `src/font_utils.py`

#### 功能清单
- ✅ **跨平台字体检测** (`check_font_installed`)
  - Windows 系统字体检测（基于 Fonts 目录）
  - macOS 系统字体检测
  - Linux 系统字体检测（基于 fc-list）
  - 支持常见中文字体映射

- ✅ **获取可用中文字体列表** (`get_available_chinese_fonts`)
  - 自动扫描系统已安装的中文字体
  - 支持 14+ 种常见中文字体检测

- ✅ **字体回退机制** (`get_fallback_font`)
  - 当指定字体不存在时提供替代方案
  - 例如：方正小标宋简体 → 黑体

- ✅ **文本分离** (`separate_text_parts`)
  - 智能识别中文、数字、英文、其他字符
  - 支持混合文本的精确分离
  - 为后续差异化格式化提供支持

### 3. 标题识别引擎 (TitleDetector)

**文件位置**: `src/title_detector.py`

#### 功能清单
- ✅ **基于样式的识别** (`detect_by_style`)
  - 识别 Word 内置样式（Title, Heading 1-4, Normal）
  - 支持中英文样式名称映射
  - 置信度：0.95（标题）/ 0.90（正文）

- ✅ **基于正则模式的识别** (`detect_by_pattern`)
  - 一级标题：一、二、三、
  - 二级标题：(一)、(二)、(三)
  - 三级标题：1. 2. 3.
  - 四级标题：(1)、(2)、(3)
  - 置信度：0.80

- ✅ **混合识别策略** (`hybrid_detect`)
  - 样式优先，正则补充
  - 自动选择最佳识别方法
  - 记录识别方法和置信度

- ✅ **置信度计算** (`calculate_confidence`)
  - 样式识别：0.95
  - 正则识别：0.80
  - 默认正文：0.90

#### 测试结果
```
✅ 基于样式的识别结果:
   关于测试的通知              → title        (置信度: 0.95)
   一、工作目标               → heading_1    (置信度: 0.95)
   这是测试正文内容...         → body         (置信度: 0.90)

✅ 基于正则模式的识别结果:
   一、工作目标               → heading_1    (置信度: 0.80)
   (二)具体措施              → heading_2    (置信度: 0.80)
   3.实施步骤                → heading_3    (置信度: 0.80)
   (4)注意事项               → heading_4    (置信度: 0.80)

✅ 混合识别结果:
   关于测试的通知              → title        (方法: style,   置信度: 0.95)
   一、工作目标               → heading_1    (方法: style,   置信度: 0.95)
   (二)具体措施              → heading_2    (方法: pattern, 置信度: 0.80)
```

### 4. 模板系统

**文件位置**: `templates/*.json`

#### 预设模板
- ✅ **公文标准格式** (`official.json`)
  - 符合 GB/T 9704-2012 党政机关公文格式标准
  - 方正小标宋简体（标题）、黑体（一级）、楷体_GB2312（二级）
  - 仿宋_GB2312（三级、四级、正文）
  - 固定行距 28.9pt，首行缩进 2 字符

- ✅ **学术论文格式** (`academic.json`)
  - 适用于学位论文、期刊文章
  - 黑体（标题）、宋体（正文）
  - 1.5 倍行距，灵活的段间距

- ✅ **商务文档格式** (`business.json`)
  - 适用于商业报告、方案书
  - 微软雅黑（全文档统一）
  - 现代简洁风格

#### 模板结构
每个模板包含：
- 页面设置（边距、纸张大小）
- 6 个层级配置（title, heading_1-4, body）
  - 字体、字号、加粗、斜体、下划线
  - 行距类型和值
  - 段前段后间距
  - 首行缩进
  - 对齐方式
  - 分页控制
- 页眉配置（可选）

#### 测试结果
