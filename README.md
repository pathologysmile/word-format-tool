# Word 文档格式自动化工具

自动将 Word 文档格式化为公文标准格式,支持自定义字体、字号、行距、段间距、页边距等参数。

## 快速开始

```bash
pip install -r requirements.txt
streamlit run app.py
```

访问 http://localhost:8501

## 功能特性

- ✅ 智能标题识别(样式+正则+手动修正)
- ✅ 完整格式配置(字体/字号/行距/段间距/页边距)
- ✅ 模板管理(保存/加载/导出)
- ✅ 公文标准预设模板

## 系统要求

- Python 3.8+
- Windows/macOS/Linux
